## 📋 8.2 FINAL PROJECT REQUIREMENTS

Untuk mendapatkan sertifikasi dan portofolio yang kuat, proyek akhir Anda wajib memenuhi kriteria berikut:

### ✅ Syarat Wajib:
1. **Multi-Model**: Menggunakan minimal 2 dari 3 jenis Machine Learning yang dipelajari.
2. **Optimization**: Model yang digunakan harus hasil optimasi (Hyperparameter Tuning).
3. **Live Demo**: Aplikasi harus dideploy dan bisa diakses secara publik (Streamlit Cloud/Railway).
4. **GitHub Repo**: Kode sumber tersimpan di repository publik dengan README yang lengkap.

### ⭐ Nilai Plus:
- Menggunakan **Docker** untuk deployment.
- Menambahkan **Data Visualization** interaktif (Plotly/Altair) pada dashboard.
- Memiliki dokumentasi API (OpenAPI/Swagger) jika menggunakan FastAPI.
